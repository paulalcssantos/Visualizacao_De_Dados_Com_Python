## 💻 Conteúdo do Notebook

O arquivo é um guia estruturado que simula um cenário real utilizando uma base de dados sobre a carreira de mulheres na área de tecnologia (*"Mulheres em Dados 2026"*).

O notebook está organizado nas seguintes etapas:
1.  **Fundamentos com Matplotlib:** Criação de gráficos básicos (linhas, barras horizontais e verticais), rotulação de eixos, títulos e controle de layout (*subplots*).
2.  **Personalização e Cores:** Aplicação de paletas customizadas (HEX), mapas de cores dinâmicos (*colormaps*) e boas práticas de exportação de imagens em alta resolução.
3.  **Análise Estatística com Seaborn:** Cruzamento de variáveis categóricas e numéricas, visualização de distribuições (Boxplots e Violinplots), matriz de correlação (Heatmaps) e estilização de temas.
4.  **Interatividade com Plotly Express:** Construção de visualizações dinâmicas com filtros nativos e detalhamento de dados ao passar o mouse (*hover*), com exportação para formato HTML.
